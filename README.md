# Practical_Machine_Learning_Assignment

This Repository was created by Benjamin Mohn, 28th of October 2018. Porpose of this Repository is to fullfill the assignment of the Practical_Machine_Learning_Assignment in the Course Course of Data Science by the Johns Hopkins University. 

## Task
The task was, to predict the quantity how barbell lifts where performt. 
Therefore data was given which was collected by 6 participants who were asked to perform the exercise in 5 different ways correctly and incorrectly. While performing the exercise the participants had accelerometer on the belt, the forearm and the arm, as wel there was an accelerometer on the dumbell.
